#### 主要技术

- Vue3
- Vite
- Pinia
- Naive-ui
- TypeScript

#### 安装

```sh
git clone https://github.com/DDDHL/holden.github.io.git
```

```sh
git clone git@github.com:DDDHL/holden.github.io.git
```

#### 运行

```sh
yarn dev
```

#### 打包

```sh
yarn build
```

#### 目录

```bash
.
├── README.md
├── dist             # 打包文件
├── index.html
├── node_modules
├── package.json
├── public
├── src
│   ├── api          # 后台接口
│   ├── assets       # 静态资源
│   ├── components   # 公用组件
│   ├── hooks        # 公用hooks
│   ├── language     # i18n配置
│   ├── pages        # 页面组件
│   ├── router       # 路由
│   ├── store        # pinia
│   ├── types        # 公用类型
│   └── utils        # 工具函数
├── prebuild.js      # 记录打包时间
└── vite.config.ts   # vite 配置

```
